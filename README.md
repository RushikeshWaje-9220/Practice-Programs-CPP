## 🔧 How to Compile & Run

**Compile:**
```bash
g++ filename.cpp -o output
```

**Run:**
```bash
./output        # Linux/Mac
output.exe      # Windows
```

**Example:**
```bash
g++ OOP.cpp -o oop
./oop
```

> Requires **G++ compiler** (part of GCC). On Windows use [MinGW](https://www.mingw-w64.org/) or [TDM-GCC](https://jmeubank.github.io/tdm-gcc/).

## 💡 Key Concepts Demonstrated

- **Inheritance** — Single, Multilevel, Multiple inheritance
- **Polymorphism** — Compile-time (Operator Overloading) and Runtime (Virtual Functions)
- **Pure Virtual Functions** — Abstract classes and interfaces in C++
- **Friend Functions** — Accessing private members from outside the class
- **this Pointer** — Implicit pointer to current object
- **const Correctness** — const variables, methods, arguments
- **Templates** — Generic programming with function/class templates
- **Dynamic Memory** — `new` and `delete` operators
- **References** — Reference variables vs pointers
- **Namespaces** — Avoiding name conflicts in large programs
